Events to listen

set-user
This event ("set-user") has to be listened on the useris verifyied and to set him online.
event-occured
This event ("event-occured") has to be listen on user's end to get updated events.
disconnect
This event ("disconnect") has to be listened on client side whenever user goes offline.
